# Getting Started

## Prerequisites

- Install the runtime dependencies for this project.
- Install `sparkify` for local docs generation: `npx sparkify --help`.

## Local Docs Workflow

1. Install dependencies.
2. Validate docs and configuration with `npx sparkify doctor`.
3. Run local preview: `npx sparkify dev`.
4. Build production docs: `npx sparkify build`.
