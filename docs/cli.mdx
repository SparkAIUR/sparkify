# CLI Reference

## `sparkify init`

Scaffold docs starter files.

## `sparkify dev`

Run local preview server with workspace rebuild on docs/python changes.

## `sparkify build`

Build static output (default `./dist`).

Useful flags:
- `--site https://<user>.github.io`
- `--base /<repo>`
- `--strict`

## `sparkify doctor`

Checks Node version, docs directory, config parsing, and OpenAPI validity.

## `sparkify export-openapi`

Exports OpenAPI from FastAPI without starting a server.

```bash
sparkify export-openapi --fastapi "app.main:app" --out ./docs/openapi.json
```
