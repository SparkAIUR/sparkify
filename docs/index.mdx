# sparkify

`sparkify` builds static docs from Mintlify-style content for GitHub Pages.

## Commands

- `sparkify init`
- `sparkify dev`
- `sparkify build`
- `sparkify doctor`
- `sparkify export-openapi`

## What it supports in v0.1.0

- Existing `docs.json` or auto-generated navigation
- MDX pages and static assets
- OpenAPI-powered API route with interactive Stoplight UI
- FastAPI schema export (`app.openapi()`)
