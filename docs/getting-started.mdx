# Getting Started

This guide covers the intended MVP workflow for `sparkify`.

## Prerequisites

- Node.js 20+
- npm
- A repository with a `docs/` folder (or run `sparkify init` once available)

## Quickstart (Planned)

```bash
npx sparkify init
npx sparkify dev
npx sparkify build --site https://<user>.github.io --base /<repo>
```

## Output

- Local preview during development
- Static build output in `dist/` for GitHub Pages deployment

## Notes

- Internal planning/spec docs remain in `refs/`.
- Public docs and dogfooding content stay in `docs/`.
