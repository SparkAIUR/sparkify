# Getting Started

## Prerequisites

- Node.js 20+
- npm
- Python 3.10-3.12 (only required for `export-openapi`)

## Quickstart

```bash
npx sparkify init
npx sparkify dev
npx sparkify build --site https://<user>.github.io --base /<repo>
```

## Config

Create `sparkify.config.json` to avoid long command flags.

```json
{
  "docsDir": "./docs",
  "outDir": "./dist",
  "base": "/my-repo"
}
```
